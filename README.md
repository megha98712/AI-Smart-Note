🧠 AI Smart Notes

AI Smart Notes is a modern and responsive note-taking web application built using HTML, CSS, and JavaScript.
It allows users to create, save, edit, and summarize notes with a simulated AI-powered assistant.

✨ Features
📝 Create and manage notes easily
💾 Save notes using Local Storage
📂 Sidebar for quick note navigation
🤖 AI-powered note summarization (simulated)
🗑️ Delete notes functionality
📱 Responsive and clean UI
⚡ Fast and lightweight


🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage API
Font Awesome Icons


🚀 How It Works
1️⃣ Create a Note

Click on the New Note button to start writing a fresh note.

2️⃣ Save Notes

Click the Save button to store notes permanently in the browser using Local Storage.

3️⃣ Load Notes

Saved notes appear in the sidebar.
Click any note to open and edit it.

4️⃣ AI Summarization

The Summarize button simulates AI behavior by generating a short summary of the written text.


Run the project

Simply open index.html in your browser.
